Installation

Please note: All Gale Resources must be installed individually. There is not a single bulk app that can be used.
Navigate to your Canvas instance and login with an administrator account.

Click the Admin shield with a key inside.

Select the Districts Name.

Click + Developer Keys.

Click + Developer Key > Click + LTI Key.


Navigate to Generate JSON for creating Developer Key for LTI 1.3 Apps

Fill out the Product Code, App Title, Description, Key*, Secret*.

*The key and secret will always be the same. If you do not know your Key or Secret please contact Gale Technical Support

Once all fields are filled out. Please click Generate JSON. This will generate a JSON response that will be used inside of the Canvas setup.

Click Copy this response to clipboard.

Return to your Canvas tab in your web browser. Fill out the Key Name (this can be the name of the database).

Change the Method dropdown to Paste JSON and paste the copied response from your clipboard that was copied.

Change the Method dropdown back to Manual and verify that all fields are filled out correctly.

If all fields are filled out correctly. Click Save at the bottom of the page.

Copy the Client ID from the Details column.

Ensure to only copy the first 16 and do not click Show Key.

Flip state toggle to green check mark.
