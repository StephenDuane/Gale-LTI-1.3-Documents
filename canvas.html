Installation

Please Note: All Gale Resources must be installed individually. There is not a single bulk app that can be used.

Navigate to your Canvas instance and login with an administrator account.

Click the Admin shield with a key inside.

Select the Districts Name.

Click + Developer Keys.

Click + Developer Key > Click + LTI Key.


Navigate to Generate JSON for creating Developer Key for LTI 1.3 Apps

Fill out Product Code, App Title, Description, Key*, Secret*.

*The key and secret will always be the same. If you do not know your Key or Secret please contact Gale Technical Support
